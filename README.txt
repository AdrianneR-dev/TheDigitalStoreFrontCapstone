For this capstone project, I am creating a digital store-front for my e-commerce beauty brand, 
Salvg (pronounced Salvage) Beauty.

Reflection:

1. I decided to use Bootstrap CSS as a beginner. I used bootstrap to style the padding and margins for the pages and buttons on each page. I am working to familiarize myself more with this CSS.

2. I am most proud of the responsive grid layout of my website. Not only does each image link to the details page of the product, each image has a lifting effect that looks like it is lifting from the page and creating a shadow when the cursor hovers over it. When the screen size changes, the number of columns of images changes as well. This creates 1 or 2 or 3 columns of images depending on the screen size.

3. The most difficult part of making my design responsive was realizing a I had repeat the same process for each webpage. I expected the entire website to follow my code but I had to tediously code my css for each navbar and each image-grid. 

4. For my about page, I used the <main> tag instead of just leaving the h1 and p tags out. For my products page, I placed the div elements in a section to incorporate semantic html, even though I could have left it in a div class.
